## Vamos Começar

Primeiro, clone o repositório

```
$ git remote add origin https://github.com/miilanez/snake-js.git
```

Depois, carregue as dependências

```
$ cd snake-js
$ npm install
``` 
Por último, mas não menos importante, execute o servidor

```
$ npm start
```
